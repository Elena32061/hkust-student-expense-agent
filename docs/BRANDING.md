# Branding Assets

This repository includes two lightweight assets for GitHub presentation:

- `assets/logo.svg`
  A simple project logo suitable for the README and repository identity.
- `assets/social-preview.svg`
  A wide preview card suitable for converting to PNG and uploading in GitHub repository social preview settings.
- `assets/social-preview.png`
  A ready-to-upload PNG export for GitHub repository social preview settings.

## Style Intent

- unofficial, not institutional
- local-first, practical, tool-like
- warm paper palette instead of corporate university branding
- abstract form and workflow motif instead of school marks or logos

## Recommended GitHub Social Preview Copy

Title:

`HKUST Student Expense Agent`

Subtitle:

`Local-first browser automation for student expense claim eForms`

Footer line:

`Snapshot the live form · Fill from explicit config · Review before submit`

## Important

Do not use HKUST official logos, crests, or protected brand marks unless you have explicit permission.
